# Tuỳ biến phong cách hình ảnh

Học cách điều chỉnh tham số sinh ảnh để tạo phong cách hình ảnh độc đáo.

---

## Tinh chỉnh tiền tố prompt

Tiền tố prompt điều khiển phong cách hình ảnh tổng thể:

```
Minimalist black-and-white illustration, clean lines, simple style
```

---

## Điều chỉnh kích thước ảnh

Mỗi kích thước phù hợp với một tình huống khác nhau:

- **1024x1024**: Vuông, phù hợp với Xiaohongshu (RED)
- **1080x1920**: Dọc, phù hợp với TikTok, Kuaishou
- **1920x1080**: Ngang, phù hợp với Bilibili, YouTube

---

## Xem trước hiệu quả

Dùng tính năng "Nghe thử phong cách" để thử các cấu hình khác nhau.

---

## Thông tin thêm

Nhiều mẹo tuỳ biến phong cách sẽ được bổ sung sau.
