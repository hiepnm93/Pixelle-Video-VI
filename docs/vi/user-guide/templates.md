# Phát triển Template

Cách tạo template video tuỳ chỉnh.

---

## Giới thiệu Template

Template video dùng HTML để định nghĩa bố cục và phong cách của khung hình video. Pixelle-Video cung cấp nhiều template có sẵn, đáp ứng nhiều kích thước và phong cách video khác nhau.

## Quy ước đặt tên template

Template tuân theo quy ước đặt tên thống nhất để phân biệt các loại:

- **`static_*.html`**: Template tĩnh
  - Không cần nội dung media do AI sinh
  - Render thuần chữ
  - Phù hợp tạo nhanh và tiết kiệm chi phí

- **`image_*.html`**: Template ảnh
  - Dùng ảnh do AI sinh làm nền
  - Gọi workflow sinh ảnh ComfyUI
  - Phù hợp nội dung cần hình minh hoạ

- **`video_*.html`**: Template video
  - Dùng video do AI sinh làm nền
  - Gọi workflow sinh video ComfyUI
  - Tạo nội dung video động giàu biểu cảm hơn

## Cấu trúc Template

Template nằm trong thư mục `templates/`, được nhóm theo kích thước:

```
templates/
├── 1080x1920/  # Dọc
│   ├── static_*.html   # Template tĩnh
│   ├── image_*.html    # Template ảnh
│   └── video_*.html    # Template video
├── 1920x1080/  # Ngang
│   └── image_*.html    # Template ảnh
└── 1080x1080/  # Vuông
    └── image_*.html    # Template ảnh
```

## Tạo template tuỳ chỉnh

### Các bước

1. Sao chép một file template có sẵn trong thư mục `templates/`
2. Sửa HTML và CSS theo ý
3. Lưu vào thư mục kích thước tương ứng với phần mở rộng `.html`
4. Dùng tên template mới trong cấu hình hoặc giao diện Web

### Biến của template

Template hỗ trợ các biến Jinja2 sau:

- `{{ title }}` — Tiêu đề video (tuỳ chọn)
- `{{ text }}` — Nội dung văn bản của cảnh hiện tại
- `{{ image }}` — Ảnh của cảnh hiện tại (nếu có)

### Ví dụ template

```html
<!DOCTYPE html>
<html>
<head>
    <style>
        body {
            width: 1080px;
            height: 1920px;
            margin: 0;
            background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
            display: flex;
            align-items: center;
            justify-content: center;
            font-family: 'Arial', sans-serif;
        }
        .content {
            text-align: center;
            color: white;
            padding: 40px;
        }
        .text {
            font-size: 48px;
            line-height: 1.6;
        }
    </style>
</head>
<body>
    <div class="content">
        <div class="text">{{ text }}</div>
    </div>
</body>
</html>
```

## Mẹo phát triển template

### 1. Kích thước tương ứng

Hãy đảm bảo kích thước `body` của template khớp với kích thước video đích:

- Dọc: `width: 1080px; height: 1920px;`
- Ngang: `width: 1920px; height: 1080px;`
- Vuông: `width: 1080px; height: 1080px;`

### 2. Trình bày văn bản

- Dùng cỡ chữ và độ cao dòng phù hợp để dễ đọc
- Thêm shadow hoặc nền cho chữ để tương phản tốt hơn
- Kiểm soát độ dài văn bản để tránh tràn

### 3. Xử lý ảnh

- Dùng `object-fit: cover` để ảnh lấp đầy
- Thêm gradient hoặc lớp phủ để cải thiện độ đọc của chữ
- Cân nhắc giải pháp dự phòng khi ảnh tải thất bại

### 4. Tối ưu hiệu năng

- Tránh CSS animation quá phức tạp
- Tối ưu kích thước ảnh nền
- Dùng font hệ thống hoặc web-safe font
