# Đóng góp

Cảm ơn bạn đã quan tâm đóng góp cho Pixelle-Video!

---

## Cách đóng góp

1. Fork repository
2. Tạo nhánh tính năng (`git checkout -b feature/AmazingFeature`)
3. Commit các thay đổi của bạn (`git commit -m 'Add some AmazingFeature'`)
4. Push lên nhánh (`git push origin feature/AmazingFeature`)
5. Tạo Pull Request

---

## Thiết lập môi trường phát triển

```bash
# Clone fork của bạn
git clone https://github.com/your-username/Pixelle-Video.git
cd Pixelle-Video

# Cài phụ thuộc cho phát triển
uv sync

# Chạy test
pytest
```

---

## Quy chuẩn mã nguồn

- Toàn bộ code và chú thích viết bằng tiếng Anh
- Tuân thủ chuẩn PEP 8
- Bổ sung test phù hợp
